What it does

🔢

Live message count

Tracks every message you send and updates the count as you go. No refresh needed.

🔔

Two-stage warnings

Alerts at 80% so you can plan, then again at 95% so you're not caught off guard.

🔴

Badge on the icon

Current count shows on the extension badge. Glanceable without opening anything.

🔄

Auto daily reset

Counter resets at midnight UTC, same as Claude's own limit window. No manual clearing.
